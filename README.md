1. CSS文件夹，

3. upload文件夹，
4. index.html

### 设置视口标签以及引入初始化样式：

1. "<meta ...>,"
2. "<link ...normalize.css>,"
3. "<link ...index.css>"

### 设置常用初始化样式：

1. body{ }

### 具体布局：

1. 固定定位的居中效果：搜索框，
2. 上图片下文字布局：局部导航栏，把主轴设成Y轴，把侧轴设成居中对齐，

4. 给前两个盒子加白色边框：导航栏，
5. 文字阴影：导航栏，
6. 背景图片靠底部居中对齐：导航栏，
